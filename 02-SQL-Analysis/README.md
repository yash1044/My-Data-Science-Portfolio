## 🛠️ SQL Technical Skillset
- **Query Optimization:** Efficient use of Common Table Expressions (CTEs) and Subqueries for readability and performance.
- **Relational Operations:** Mastery over complex Inner, Left, and Full Outer Joins across multiple tables.
- **Analytical Functions:** Expertise in Window functions like `RANK()`, `DENSE_RANK()`, and `ROW_NUMBER()`.
- **Database Design:** Building normalized schemas from scratch with Primary and Foreign Key constraints.
- **Business Logic:** Implementing conditional logic using `CASE` statements and advanced data filtering.

## 📂 Folder Structure
Each project folder includes:
1. `Queries.sql`: The complete script with documented analytical steps and 50+ queries.
2. `Dataset.csv`: (Where available) Raw data used for the analysis.
3. `README.md`: Specific project documentation with business insights and objectives.
