# 🐍 Python Fundamentals

This repository contains a collection of practice files covering the core concepts of Python programming. Building a strong foundation in Python is the first and most crucial step in my Data Science and Machine Learning journey.

## 📁 Directory & Topics Covered

Each file focuses on specific programming logic and Pythonic syntax:

1.  **01_Python_Variables_and_Indexing:** Understanding variables, memory allocation, and the basics of sequence indexing.
2.  **02_Python_List_Methods_and_Slicing:** Deep dive into List data structures, slicing techniques, and essential list methods.
3.  **03_Python_List_Advanced_Methods:** Exploring complex list manipulations and advanced built-in functions.
4.  **04_Python_TypeCasting_Input_and_Strings:** Handling user inputs, converting data types (Typecasting), and string formatting.
5.  **05_Python_Set_Operations_and_Methods:** Working with Sets and performing mathematical operations like Union and Intersection.
6.  **06_Python_Dictionary_Methods_and_Operations:** Managing Key-Value pairs and dictionary-specific methods.
7.  **07_Python_NestedDict_and_Operators:** Handling nested data structures and using various Python operators (Arithmetic, Logical, etc.).
8.  **08_Python_Conditions_and_Identity_Operators:** Implementing decision-making logic using If-Else and Identity operators.
9.  **09_Python_While_and_For_Loops:** Mastering iteration and repetitive task execution using loops.
10. **10_Python_Functions_and_Lambda_Expressions:** Writing reusable code blocks with Functions and Anonymous (Lambda) expressions.

## 🛠️ Tech Stack & Tools
* **Language:** Python 3.x
* **Environment:** Jupyter Notebook (.ipynb)
