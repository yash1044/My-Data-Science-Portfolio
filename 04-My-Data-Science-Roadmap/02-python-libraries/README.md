# 📚 Python Data Science Libraries

This folder serves as a comprehensive guide to the core libraries used in the Python Data Science ecosystem. It covers everything from raw data manipulation and cleaning to advanced statistical visualization.

## 📁 Folder Structure

* **/datasets:** Contains all the CSV/Excel files used as data sources for the notebooks.
* **01 to 08 Files:** Sequential practice notebooks for mastering the data stack.

## 📝 Topics & Notebooks Overview

1.  **01_Pandas_Data_Selection_and_Indexing:** Mastering DataFrame navigation using `.loc`, `.iloc`, and conditional filtering.
2.  **02_Pandas_Data_Cleaning_and_Imputation:** Techniques for handling missing values (NaN), removing duplicates, and data type correction.
3.  **03_Pandas_Grouping_Merging_and_Sorting:** Advanced data aggregation using `groupby`, joining multiple datasets, and custom sorting.
4.  **04_Pandas_Concatenation_and_CrossTabulation:** Combining datasets vertically/horizontally and performing frequency analysis with `crosstab`.
5.  **05_NumPy_Arrays_and_Mathematical_Operations:** Vectorized computing, N-dimensional arrays, and high-performance math functions.
6.  **06_NumPy_Advanced_Indexing_and_Reshaping:** Complex array manipulation, broadcasting, and reshaping techniques for ML models.
7.  **07_Matplotlib_Data_Visualization_Basics:** Creating fundamental plots like Line, Bar, and Scatter to understand data distributions.
8.  **08_Seaborn_Advanced_Statistical_Visualization:** Building complex charts like Heatmaps, Pairplots, and Violin plots for deep statistical insights.

## 🛠️ Tech Stack
* **Data Analysis:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook (.ipynb)
