# 🚀 Data Science & Analytics Roadmap

Welcome to my comprehensive Data Science learning journey. This repository is a structured collection of my technical skills, ranging from core Python programming to advanced Machine Learning and Data Visualization.

## 📌 Roadmap Overview

I have organized my learning into 5 key modules, each focusing on a critical aspect of the Data Science lifecycle:

### 1️⃣ [Python Fundamentals](./01-python-fundamentals)
* **Goal:** Building a strong logic and programming foundation.
* **Topics:** Data types, loops, conditional statements, and basic problem-solving.

### 2️⃣ [Data Manipulation with Python Libraries](./02-python-libraries)
* **Goal:** Mastering tools for data handling and analysis.
* **Topics:** In-depth practice of **Pandas** for dataframes and **NumPy** for numerical computations.

### 3️⃣ [Machine Learning Implementation Lab](./03-machine-learning-algorithms)
* **Goal:** Applying algorithms to real-world datasets.
* **Structure:** - **Phase 01:** Preprocessing & Regression (Outliers, Scaling, Encoding).
    - **Phase 02:** Supervised Learning (Classification - SVM, KNN, NB, Decision Trees).
    - **Phase 03:** Unsupervised Learning (K-Means Clustering).

### 4️⃣ [SQL for Data Analytics](./04-sql-data-analytics)
* **Goal:** Mastering data extraction and database management.
* **Topics:** Joins, Aggregations, Subqueries, and Conditional Logic (`CASE WHEN`).

### 5️⃣ [Tableau Visualizations](./05-tableau-visualizations)
* **Goal:** Transforming raw data into visual insights.
* **Topics:** Interactive charts, Dual-axis graphs, Groups, Sets, and Calculated Fields.

---

## 🛠️ Technical Skill Set

* **Languages:** Python (Pandas, NumPy, Scikit-Learn), SQL.
* **ML Algorithms:** Regression, KNN, SVM, Naive Bayes, Random Forest, K-Means.
* **Tools:** Tableau, Jupyter Notebook, MySQL/PostgreSQL.
* **Data Engineering:** Outlier Detection (Z-Score/IQR), Feature Scaling, Encoding.
