# 📊 Tableau Data Visualizations

This folder represents my journey of mastering data visualization in Tableau. I have structured my learning into three progressive modules, moving from basic chart creation to advanced analytical techniques.

## 📁 Workbook Overview

### 1️⃣ [01_Tableau_Basic_Visualizations](./01_Tableau_Basic_Visualizations.twb)
* **Focus:** Fundamental chart types and formatting.
* **Key Skills:** Creating Bar Charts, Line Graphs, and Pie Charts. Understanding the Tableau interface, dimensions, and measures.

### 2️⃣ [02_Tableau_Groups_Sets_and_Filters](./02_Tableau_Groups_Sets_and_Filters.twb)
* **Focus:** Data organization and dynamic analysis.
* **Key Skills:** - **Groups & Sets:** Organizing data points for better comparison.
    - **Filters:** Implementing interactive filters to slice data.
    - **Calculated Fields (CF):** Writing custom formulas to derive new insights.

### 3️⃣ [03_Tableau_Advanced_Charts_DualAxis](./03_Tableau_Advanced_Charts_DualAxis.twb)
* **Focus:** Advanced visual analytics.
* **Key Skills:** - **Dual-Axis Charts:** Comparing two different measures on the same graph.
    - **Geographic Mapping:** Visualizing data across regions.
    - **Advanced Formatting:** Creating sophisticated visuals for complex datasets.

## 🛠️ Tech Stack
* **Tool:** Tableau Desktop / Tableau Public
* **Data Sources:** Excel, CSV
* **Key Techniques:** Data Blending, Sets, Groups, Parameters, and Calculated Fields.
