# 🗄️ SQL for Data Analytics

SQL is the fundamental tool for data extraction and transformation. This folder contains my structured practice scripts, covering everything from basic data retrieval to complex joins and conditional logic.

## 📁 Repository Structure & Topics

1. **01_SQL_Fundamentals_and_Data_Retrieval:**
   - Basics of `SELECT`, `FROM`, and `DISTINCT`.
   - Understanding data types and simple data fetching.

2. **02_SQL_Advanced_Filtering_and_Date_Functions:**
   - Filtering data using `WHERE`, `AND`, `OR`, and `IN` operators.
   - Working with Date & Time functions for time-series analysis.

3. **03_SQL_Aggregations_and_Data_Manipulation:**
   - Mastering aggregate functions: `SUM()`, `AVG()`, `COUNT()`, `MIN()`, and `MAX()`.
   - Data grouping using `GROUP BY` and filtering groups with `HAVING`.

4. **04_SQL_Subqueries_Pagination_and_Joins:**
   - Writing nested queries (Subqueries) for complex data extraction.
   - Understanding table relationships through `JOIN` (Inner, Left, Right).
   - Implementing pagination using `LIMIT` and `OFFSET`.

5. **05_SQL_Table_Management_and_Conditional_Logic:**
   - Data Definition Language (DDL) like `CREATE`, `ALTER`, and `DROP`.
   - Using `CASE WHEN` statements for implementing conditional logic within queries.

## 🛠️ Skills Demonstrated
* Relational Database Management (RDBMS)
* Data Cleaning & Transformation via SQL
* Complex Query Optimization
* Business Logic Implementation
