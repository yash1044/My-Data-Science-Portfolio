In this final phase of the Machine Learning implementation lab, the focus shifts to **Unsupervised Learning**. Unlike previous phases where the models learned from labeled data, here the algorithms explore unlabeled datasets to discover inherent groupings and structures.

## 📁 Implementation Overview

1. **12_ML_KMeans_Clustering_Unsupervised_Learning:** - Implementation of the K-Means algorithm to group data points based on feature similarity.
   - Performing Exploratory Data Analysis (EDA) to understand cluster distributions.

## 💡 Key Concepts Explored
* **Centroids & Clusters:** Understanding how the model iteratively assigns data points to the nearest cluster center.
* **The Elbow Method:** Using Within-Cluster Sum of Squares (WCSS) to find the optimal number of clusters (K).
* **Data Segmentation:** Practical application of clustering in scenarios like customer segmentation and market analysis.
