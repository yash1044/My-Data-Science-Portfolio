# 🎯 Phase 02: Supervised Learning (Classification)

After mastering data preprocessing in Phase 01, this stage focuses on building predictive models to classify data into distinct categories. These algorithms are essential for solving real-world problems like risk assessment, medical diagnosis, and spam filtering.

## 📁 Algorithms & Notebooks Overview

1. **07_Logistic_Regression:** Predicting binary outcomes (e.g., Insurance Purchase Prediction) using categorical probability.
2. **08_Support_Vector_Machine (SVM):** Utilizing hyperplanes to find the most effective boundaries between different data classes.
3. **09_Decision_Tree_Classification:** Implementing rule-based hierarchical models for complex decision-making logic.
4. **10_KNN_Algorithm:** A proximity-based classification technique that categorizes data points based on their neighbors.
5. **11_Naive_Bayes_Spam_Detection:** Leveraging probabilistic methods (Bayes' Theorem) for efficient text classification and spam detection.

## 📊 Model Evaluation Metrics
To ensure high-performance models, I have evaluated these algorithms using:
* **Confusion Matrix:** To track True/False Positives and Negatives.
* **Accuracy Score:** To measure the overall reliability of the model.
* **Classification Report:** Detailed analysis of Precision, Recall, and F1-Score.

## 🛠️ Tech Stack
* **Library:** Scikit-Learn (Sklearn)
* **Data Handling:** Pandas, NumPy
* **Visualization:** Seaborn, Matplotlib
