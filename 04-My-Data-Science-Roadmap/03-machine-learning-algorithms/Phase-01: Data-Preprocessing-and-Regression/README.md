# 🛠️ Phase 01: Data Preprocessing & Regression

This phase focuses on the most critical part of Data Science: **Preparing the data.** Before building complex models, it is essential to handle noise, outliers, and ensure the data is in the right format.

## 📁 Topics & Notebooks Overview

1.  **01_Simple_Linear_Regression:** Predicting a dependent variable based on a single independent variable (e.g., House Price vs. Square Feet).
2.  **02_Multiple_Linear_Regression:** Handling multiple features to improve prediction accuracy.
3.  **03_Feature_Encoding:** Converting categorical data into numerical format using techniques like One-Hot Encoding and Label Encoding.
4.  **04_Feature_Scaling:** Normalizing data distributions to ensure all features contribute equally to the model.
5.  **05_Outlier_Detection (Z-Score):** Using statistical standard deviations to identify and handle data anomalies.
6.  **06_Outlier_Detection (IQR Method):** Using the Interquartile Range to detect and treat outliers for robust modeling.

## 💡 Key Learnings
* Understanding the impact of outliers on model performance.
* Implementing mathematical methods (Z-Score & IQR) for data cleaning.
* Managing the bias-variance tradeoff in Regression models.

## 🛠️ Tools Used
* **Libraries:** Scikit-Learn, Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
