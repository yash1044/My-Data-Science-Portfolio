# 🤖 Machine Learning Implementation Lab

This repository represents a structured journey through the world of Machine Learning. It covers the entire lifecycle of an ML project—from data cleaning and feature engineering to deploying supervised and unsupervised models.

## 📂 Project Structure

To make the learning process clear and organized, I have divided the implementation into three distinct phases:

### [🔹 Phase 01: Data Preprocessing & Regression](./Phase-01: Data-Preprocessing-and-Regression)
* **Focus:** Data cleaning, handling outliers (Z-Score & IQR), and feature scaling.
* **Models:** Simple and Multiple Linear Regression for continuous value prediction.

### [🔹 Phase 02: Supervised Learning (Classification)](./Phase-02: Supervised-Learning-Classification)
* **Focus:** Predictive modeling for categorical data.
* **Algorithms:** Logistic Regression, SVM, Decision Trees, KNN, and Naive Bayes for spam/risk detection.

### [🔹 Phase 03: Unsupervised Learning](./Phase-03: Unsupervised-Learning)
* **Focus:** Finding hidden patterns in unlabeled data.
* **Technique:** K-Means Clustering and the "Elbow Method" for optimal cluster selection.

## 🛠️ Tech Stack & Skills
* **Core Library:** Scikit-Learn
* **Data Manipulation:** Pandas & NumPy
* **Visual Insights:** Matplotlib & Seaborn
* **Concepts:** Feature Encoding, Outlier Treatment, Model Evaluation (Confusion Matrix, Accuracy).

---
*Goal: To build a solid foundation in machine learning by implementing core algorithms from scratch using real-world datasets.*
