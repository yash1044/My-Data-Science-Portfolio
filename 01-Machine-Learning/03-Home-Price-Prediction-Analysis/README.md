# 🏠 Home Price Prediction using Linear Regression

## 📌 Project Overview
This project focuses on predicting real estate prices based on various features such as square footage, number of bedrooms, and location. It demonstrates the application of fundamental regression techniques to solve real-world pricing problems.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Algorithm:** Linear Regression

## 📊 Key Workflow
- **Data Exploration:** Analyzed the relationship between different features and the target price.
- **Pre-processing:** Handled missing values, performed feature scaling, and dealt with categorical variables.
- **Model Building:** Implemented a Linear Regression model to establish a baseline for price prediction.
- **Evaluation:** Used metrics like Mean Absolute Error (MAE) and R-squared to assess model accuracy.

## 📂 Project Structure
- 📄 `Home_Price_Prediction_Linear_Regression.ipynb` : Full implementation of the regression model.
- 📊 `Housing_Data.csv` : Dataset used for training and testing.
- 📝 `README.md` : Project documentation.
