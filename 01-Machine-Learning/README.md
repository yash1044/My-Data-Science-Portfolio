
# 🤖 Machine Learning Portfolio

This folder contains a diverse collection of 9 Machine Learning projects, covering Supervised and Unsupervised learning techniques. Each project focuses on solving specific real-world problems using Python and various ML algorithms.

## 📊 Project Summary Table

| No. | Project Name | Type | Algorithm Used | Key Skills |
|:---:|:---|:---:|:---:|:---|
| 01 | **SkyPrice** | Regression | Random Forest | Feature Engineering, Price Prediction |
| 02 | **CreditShield** | Classification | KNN / Logistic | Risk Assessment, Scaling |
| 03 | **Home Price** | Regression | Linear Regression | Data Cleaning, Baseline Modeling |
| 04 | **FlyHigh** | Classification | Decision Tree | EDA, Service Quality Analysis |
| 05 | **Titanic Survival** | Classification | Logistic Regression | Imputation, ROC/AUC Analysis |
| 06 | **LendLogic** | Classification | Naive Bayes | Financial Automation, Encoding |
| 07 | **InsurePredict** | Regression | Linear Regression | Error Metrics (MSE, RMSE), BMI Analysis |
| 08 | **StayCheck** | Classification | Random Forest | Ensemble Learning, Cancellation Trends |
| 09 | **HeartSync** | Clustering | K-Means & Hierarchical | Unsupervised Learning, Silhouette Score |

## 🛠️ Technical Skillset
- **Data Pre-processing:** Automated Mean/Mode Imputation, Label & One-Hot Encoding, StandardScaler, Robust Outlier Detection.
- **Algorithms:** Linear & Logistic Regression, Decision Trees, Random Forest (Ensemble), K-Nearest Neighbors, Naive Bayes, K-Means Clustering.
- **Evaluation Metrics:** R2 Score, MSE, Accuracy, Precision-Recall, F1-Score, Confusion Matrix, Silhouette Score, ROC Curve.
- **Libraries:** Pandas, NumPy, Scikit-learn, Scipy, Matplotlib, Seaborn.

## 📂 Folder Structure
Each project folder contains:
1. `Notebook.ipynb`: Complete documented code.
2. `Dataset.csv`: Raw or processed data used for the model.
3. `README.md`: Detailed explanation of that specific project.
