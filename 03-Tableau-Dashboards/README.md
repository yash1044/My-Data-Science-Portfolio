This repository contains interactive dashboards and data stories built using Tableau. These projects demonstrate my ability to transform complex raw data from various industries (Gaming and E-commerce) into clear, actionable visual insights that drive business decisions.

## 🛠️ Tableau Technical Skillset
- **Data Integration:** Connecting to and cleaning diverse datasets for visualization.
- **Advanced Calculations:** Expertise in creating Calculated Fields and using LOD Expressions (FIXED, INCLUDE).
- **Interactive Design:** Implementing Dashboard Actions (Filters, Highlights), Parameters, and Tooltips for better UX.
- **Data Storytelling:** Building sequential stories to explain trends and pinpoint business opportunities.
- **Visual Standards:** Creating clean, corporate-standard layouts with intuitive color palettes.

## 📂 Folder Structure
Each project folder contains:
1. `.twbx` File: The packaged Tableau workbook.
2. `Preview.png`: A high-resolution screenshot of the dashboard for quick viewing.
3. `README.md`: Detailed breakdown of the project objectives and data insights.
