# 🛒 Blinkit Operations: Quick Commerce Insights (Tableau)

## 📌 Project Overview
This project is an end-to-end data visualization study of **Blinkit’s** operational data. The goal is to analyze sales trends, item performance, and outlet efficiency to provide actionable business insights using **Tableau**.

## 📊 Key Insights & Metrics
- **Total Sales Analysis:** Monitoring overall revenue and average sales per order.
- **Product Performance:** Identifying top-selling categories (e.g., Snacks, Dairy, Fruits).
- **Outlet Segment:** Analyzing sales based on outlet size and location (Tier 1, 2, 3 cities).
- **Customer Ratings:** Correlating product types with customer satisfaction scores.

## 🛠️ Tools Used
- **Tableau Desktop** (for Dashboard & Storytelling)
- **Dataset:** Blinkit Operational Sales Data
